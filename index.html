API Name API Description
1. Earthquake Returns a list of earthquakes, ordered by magnitude
2. Weather Stations with most recent Weather Observation Returns a list of weather stations with the most recent weather observation
3. Postal Code Search Returns a list of postalcodes and places for the lat/lng query as xml document. The result is sorted by distance. For Canada the FSA is returned (first 3 characters of full postal code)
